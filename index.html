프로젝트 개요

K사 그룹 전체 IT 인프라에 대한 통합 보안 검증 및 모의해킹 프로젝트

주요 업무
  • 웹/모바일 애플리케이션 취약점 진단 및 모의해킹
  • 내부 기준 진단 및 CVE 취약점 확인
  • 담당자 문의 응대 및 협업을 통한 취약점 조치
  • PL 부재 시 PL 대무 역할 수행
주요 성과
  • 타 검증원과의 크로스체크를 통한 월별 업무평가 점수 향상
  • 기존 검증 기준에 없던 신규 취약점 발견:
    • Unity 파일 내 Command Injection
    • AWS CLI를 통한 S3 버킷 서버 접근 취약점
    • AI Prompt Injection
    • Geoserver SQL Injection
  • Critical 취약점 높은 발견율 유지
사용 기술

Burp Suite Python AWS CLI Frida
